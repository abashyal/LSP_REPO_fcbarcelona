# Question 4 – Reflection on AI in Programming

Before this class, I had never used AI tools for any of my coursework. I was honestly a little hesitant at first because I was not sure if it would actually help me learn or just do the work for me. What I liked about this course is that it was very AI oriented — we were encouraged to use these tools and reflect on them, which made me more comfortable experimenting with them throughout the semester. I found that AI was most useful when I already had some understanding of what I was trying to do, because then I could actually tell if the output made sense or not.

Going forward, I plan to use AI on future projects where I can. It is especially helpful for getting started on something unfamiliar or for checking your work, but you still need to understand the material yourself to use it effectively. This class taught me that AI is a tool, not a replacement for actually knowing what you are doing.
